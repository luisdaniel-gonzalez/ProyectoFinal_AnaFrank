# PROYECTO FINAL: EL DIARIO DE ANA FRANK

Proyecto inspirado en el hito historico de Ana Frank. El jugador avanza por tres niveles que representan momentos clave de su vida, aplicando físicas simples y agentes autónomos.


## Niveles:

**Nivel 1 - El escondite**
Moverse sin hacer ruido.
- Vista: Cenital
- Física: Colisiones y velocidad según superficie
- Agente: Guardia que reacciona a sonidos

**Nivel 2 - Cartas desde el ático**
Ordenar fragmentos del diario disparando parábolas.
- Vista: Frontal fija
- Física: Movimiento parabólico
- Agente: Anotador que ofrece pistas

**Nivel 3 - Entre las sombras**
Escapar evitando las luces oscilantes.
- Vista: Lateral
- Física: Movimiento pendular y circular
- Agente: Peter, aprende tiempos de oscilación
